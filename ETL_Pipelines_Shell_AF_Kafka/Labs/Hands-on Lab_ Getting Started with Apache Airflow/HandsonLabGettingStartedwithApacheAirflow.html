Hands-on Lab: Getting Started with Apache Airflow

Objectives

After completing this lab you will be able to:

  • Start Apache Airflow.
  • Open the Airflow UI in a browser.
  • List all the DAGs.
  • List the tasks in a DAG.
  • Explore a DAG in the UI.
